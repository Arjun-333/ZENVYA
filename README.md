# ZENVYA | Neural Commerce Intelligence

ZENVYA is a high-fidelity eCommerce intelligence engine designed to intercept marketplace flux and optimize commerce acquisition. It transforms the traditional "shopping" experience into a tactical advantage by leveraging neural sentiment analysis and cross-platform price matching.

## Core Vision
In an era of artificial price inflation and review manipulation, ZENVYA acts as a Commerce Concierge, providing users with verified market intelligence and capital optimization strategies.

## Key Systems

### 1. The Intelligence Portal
A cinematic entry point featuring 3D tech artifacts and real-time cursor tracking, signifying the system's scanning logic as it prepares to intercept the market.

### 2. Market Flux (Intelligence Engine)
*   Neural Matching: Fuzzy-logic deduplication that groups identical products across Amazon, Flipkart, and Croma.
*   Integrity Quotient: Real-time sentiment intercepts using a DistilBERT ML pipeline to verify product quality.
*   Decision Badging: Immediate tactical advice via Value King, Price Floor, and Risky Flux markers.

### 3. Bespoke Atelier (Personal Stylist)
A high-fidelity neural dressing room. Upload your profile to initiate Aesthetic Mapping, allowing the system to curate intercepts that align with your structural profile and biological color palette.

### 4. Oracle Concierge (Conversational Advisor)
A conversational interface that handles natural language queries, budget tracking, and financial stewardship.

### 5. Multi-Theme Integrity
Supports the Void Dark (Neon-Tech) and Antique Parian Light (Reference-based parchment) aesthetics, ensuring a premium visual experience in any environment.

## Technical Stack
- Frontend: Next.js 14, Tailwind CSS, Framer Motion, Spline, GSAP.
- Backend: FastAPI, Python 3.10.
- ML Layer: HuggingFace Transformers (DistilBERT), FuzzyWuzzy matching.
- Data Layer: High-speed internal Neural Store with Playwright scraping infrastructure.

## Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+

### Installation

1. Clone the Intercept
   ```bash
   git clone https://github.com/Arjun-333/ZENVYA.git
   cd ZENVYA
   ```

2. Initialize the Neural Engine (Backend)
   ```bash
   cd backend
   pip install -r requirements.txt
   uvicorn api.main:app --reload
   ```

3. Initialize the Pulse (Frontend)
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
